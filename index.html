Quem eu sou.

Eu sou engenheiro cartógrafo graduado pelo Instituto Militar de Engenharia (IME - 2016) com especialização em Engenharia de Software pela Pontíficia Universidade Católica (PUC - MG). Atuo na área de produção cartográfica no 3º Centro de Geoinformação (3º CGEO) situado na cidade de Olinda. Meu trabalho como engenheiro envolve a produção de cartas e mapas para o Exército Brasileiro, geoprocessamento, levantamentos topográficos e desenvolvimento de soluções de produção de geoinformação.

Habilidades e Competências.

Programação em Python

Familiaridade com a linguagem python há pelo menos 5 anos tendo desenvolvido ferramentas para o software QGIS utilizando a linguagem, análise exploratória de dados, implementação de algoritmos de machine learning e projetos de desenvolvimento web.

Postgres

Experiência com gerenciamento de banco de dados com o sistema Postgres. Já participei de projetos em que foi necessário integrar bancos de dados geoespaciais aos softwares QGIS e ArcGIS, bem como já desenvolvi bancos de dados para aplicações web e criei scripts na linguagem SQL para consultas a banco de dados de forma a otimizar operações de interesse do cliente.

Engenharia Cartográfica

Engenheiro Cartógrafo de formação, atuo na produção cartográfica elaborando mapas temáticos, cartas topográficas e utilizando técnicas de geoprocessamento para entregar soluções que melhor atendem as necessidades do cliente.
